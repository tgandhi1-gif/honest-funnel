# The Honest Funnel

> The funnel was taking credit for sales it never earned. I rebuilt attribution from scratch so every channel told the truth.

The question: Which channels actually earned our conversions, and which were just taking credit?
The result: Reclaimed roughly a quarter of spend hiding behind false last click credit.

## Run
    pip install -r requirements.txt
    python main.py

## Stack
SQL, dbt, Python (this demo runs a Markov removal effect model in pandas).
